# begin

environment:

- python 3.10

## install environment

```
conda create -n yolo python=3.10
```

```
conda activate yolo
```

install as edit mode

```
pip install -e . -i https://mirrors.aliyun.com/pypi/simple/
```

## run test

```
pytest tests/test_python.py
```
